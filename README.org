* Emacs config
** Quick start

Install:

- git
- [[https://nodejs.org/en/download][npm]]
- [[https://www.python.org/downloads/][python]] and [[https://docs.astral.sh/uv/getting-started/installation/][uv]]
- [[https://github.com/BurntSushi/ripgrep#installation][ripgrep]]
- [[https://github.com/joaotavora/rassumfrassum][rassumfrassum]]

Clone this repo into =~/.emacs.d= (or =%APPDATA%/.emacs.d= on Windows).

** Installation

This config assumes Emacs 30+ (built-in tree-sitter).

**** Build

#+begin_src bash
git clone https://git.savannah.gnu.org/git/emacs.git
cd emacs
./autogen.sh

./configure\
    --prefix=/usr/local \
    --with-native-compilation=aot \
    --with-pgtk \
    --with-tree-sitter \
    --with-cairo \
    --with-harfbuzz \
    --with-sqlite3 \
    --with-xml2 \
    --with-gnutls \
    --with-modules \
    --with-gif \
    --with-png \
    --with-jpeg \
    --with-tiff \
    --with-rsvg \
    --with-webp \
    --with-mailutils \
    --with-xwidgets

make -j$(nproc)
sudo make install
#+end_src

*** Windows

**** Install from site

Go to [[https://alpha.gnu.org/pub/gnu/emacs/pretest/windows/emacs-31/]] and download the 'emacs-31.0.90.zip', then run 'bin/runemacs.exe'

***** Disable the Smart App Control:

Go to:

Settings → Privacy & security → Windows Security → App & browser control → Smart App Control

** Windows notes

- Git is expected at =C:/Program Files/Git/usr/bin= (added to PATH in =init.el=).
- =bash-completion= uses WSL when available.

- Set ctrl to caps
  #+begin_src pwsh
  $hexified = "00,00,00,00,00,00,00,00,02,00,00,00,1d,00,3a,00,00,00,00,00".Split(',') | % { "0x$_"};
  $kbLayout = 'HKLM:\System\CurrentControlSet\Control\Keyboard Layout';
  New-ItemProperty -Path $kbLayout -Name "Scancode Map" -PropertyType Binary -Value ([byte[]]$hexified);
  #+end_src
